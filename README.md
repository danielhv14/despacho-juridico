# ⚖️ Despacho Jurídico — Sistema de Gestión

CRUD completo para gestión de despacho jurídico desarrollado con Python y MySQL.

## 🛠️ Tecnologías
- Python 3.12
- MySQL (XAMPP)
- CustomTkinter
- mysql-connector-python

## 📋 Módulos
- Clientes
- Abogados
- Casos / Expedientes
- Audiencias
- Honorarios
- Documentos
- Notas

## ⚙️ Requisitos
- Python 3.12+
- XAMPP con MySQL corriendo
- Librerías: ver requirements.txt

## 🚀 Instalación
1. Clonar el repositorio
2. Crear entorno virtual: `python3 -m venv venv`
3. Activar: `source venv/bin/activate`
4. Instalar librerías: `pip install -r requirements.txt`
5. Importar base de datos desde `database.sql`
6. Ejecutar: `python3 app.py`
